Как бесплатно залить сайт на GitHub Pages

Регистрация на GitHub

Для создания регистрации перейдите по ссылке https://github.com/signup.

Создание репозитория

Для создания репозитория перейдите по ссылке https://github.com/new.

Важно, чтобы название репозитория было в виде «username.github.io», где username — имя вашего аккаунта на GitHub. В моем примере это будет https://danilkurmayev.github.io/aitstudent

Загрузка файлов

Репозиторий создан,

Нажмите кнопку uploading an existing file.

Теперь нужно загрузить файлы. Для этого нужно в своей папке с проектом, вабрать файлы и папки CTR+A и перетащить их в репозиторий.

Добавим комментарий к нашему коммиту и нажмём кнопку «Commit changes».

Теперь переходим во вкладку«Settings».

Во вкладке «Settings» выберите вкладку «Pages». И здесь в Branch выбираем «main» и жмем кнопку «Save» после этого начнется публикация сайта

За статусом публикaции можно посмотреть во вкладке «Actions»

Как закончиться публикация, «pages-build-deployment» станет с зеленой галочкой

Затем переходим во вкладку «Code» и выбираем пункт меню «Deployments»

Появится окно «Latest deployments from github-pages» где вам нужно нажав на три точки выбрать «view workflow run»

Появится окно «pages build and deployment» где вам нужно нажать на ссылку вашего сайта https://danilkurmayev.github.io/aitstudent/»
