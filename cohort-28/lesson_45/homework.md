# Задача

```
Смоделировать предметную область "Телевидение"

Класс Телевизор (TV)
    - Список каналов (channels)

Класс Канал (Channel)
    - Название (name)
    - Номер (number)
    - Список телепередач (programms)
    
Класс Телепередача
    - Название (title)
    
Класс Пульт (RemoteController)
    - Телевизор
    - метод включитьКанал(номер канала)
    
Когда у пульта вызывают метод включитьКанал, то канал с номером `номер канала`
показывает свою случайную передачу.

* Можете предусмотреть выключение телевизора и т.д.

Отправить на проверку:
- TV.java
- Channel.java
- Program.java
- RemoteController.java
- Main.java - с демонстрацией работы
- Вывод консоли (что консоль выводит)
```
